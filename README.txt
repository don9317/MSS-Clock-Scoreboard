# My Sport Space Clock, Scoreboard & Signage v5.0

GitHub Release Version

## Includes

- Clock display
- Scoreboard display
- Dual-screen Controller + Display
- Home/Away team names
- Score controls
- Game clock
- Clock presets and corrections
- Timeout clock
- Timeout pauses game clock
- Timeouts per game
- Team fouls
- Period tracking
- Horn / Test Horn
- Sponsor logo
- Facility logo
- Ad slides
- Facility announcements
- Appearance controls
- Background controls
- Exit Display / Exit Program
- GitHub launcher page

## Start locally

Open `Start-Here.html` or `index.html`, then click **Open Controller + Display**.

## GitHub Pages

Use `index.html` as the main launcher page.

## Display setup

1. Open Controller + Display.
2. Keep Controller on the laptop.
3. Move Display to the TV using Windows + Shift + Arrow.
4. Press F11 on the Display window.
5. Click Enable Horn Sound once on the Display window.
6. Use Test Horn on Controller to confirm sound.

## Notes

- In Clock mode, the court/location appears once directly above the time.
- Starting a timeout automatically pauses the game clock.
- Use Reset Saved Settings on the launcher page if old local settings cause issues.
